## 🚀 整合后的开发流程

### 1. 快速启动 (2分钟)
```bash
# 进入项目目录
cd basic

# 一键启动完整开发环境
./scripts/web-dev-environment.sh start
```

### 2. 访问应用
- **🌐 前端应用**: http://localhost:10086
- **🛠️ 开发工具**: http://localhost:10086/pages/dev-tools/index
- **🔗 API Gateway**: http://localhost:8080
- **🤖 AI Service**: http://localhost:8206

### 3. 开发流程
1. **启动开发环境** → 一键启动所有服务
2. **访问开发工具** → 内置调试和测试界面
3. **运行健康检查** → 确保所有服务正常
4. **开始开发** → 修改代码自动热加载
5. **实时测试** → 使用开发工具进行功能测试
